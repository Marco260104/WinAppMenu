📌 Objetivo General

Diseñar e implementar una aplicación de escritorio en Visual Studio con Windows Forms en el lenguaje C# que permita gestionar de manera sencilla los productos (mochilas y bolsos) de una tienda, mediante formularios interactivos, controles visuales y archivos XML como almacenamiento.

🎯 Temática

La aplicación simula el funcionamiento de una tienda que comercializa mochilas y bolsos con distintas categorías, precios, colores, usos, garantías y proveedores. La interfaz permite agregar, buscar, editar, eliminar productos y generar reportes con base en diversos filtros.

🛠 Funcionalidades Principales

  • 🔐 Login con validación.
  • 📋 Menú de navegación con opciones organizadas por módulos
  • ➕ Agregar productos con sus características (nombre, precio, garantía, uso, proveedor, etc.)
  • 🔎 Búsqueda individual por nombre o código
  • 📝 Edición y eliminación de productos existentes
  • 📊 Generación de reportes (por color, precio, garantía, uso, proveedor...)
